# Iced Countdown

A simple countdown timer app built with Rust and iced.

## About

Iced Countdown is a small desktop countdown app created as a learning project while exploring GUI development with Rust and the `iced` framework.

## Features

- Set hours, minutes, and seconds
- Start, pause, and reset the countdown
- Minimal circular timer interface
- Custom-styled buttons

## Purpose

This project is mainly a training project for learning:

- Rust GUI development
- The `iced` widget system
- App state management
- Custom styling
- Layout organization
- Interactive desktop UI design
